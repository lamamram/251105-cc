# utiliser le prompt

## choix du modèle

* Sonnet-4: tâches courantes (par défaut)
* pour utiliser Opus-4: tâches plus lourdes
  + `claude --model opus-4` dans la commande
  + `/model opus-4` dans le prompt
* Haiku-4 (SML): tâches légères
